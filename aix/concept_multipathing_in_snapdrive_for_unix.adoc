= Multipathing in SnapDrive per UNIX
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapDrive per UNIX supporta il multipathing. SnapDrive per UNIX non supporta l'utilizzo di alias per i dispositivi creati utilizzando `mknod` e/o. `rendev` comandi.
